# Ann-Project
# 🏦 Bank Customer Churn Prediction using Artificial Neural Network (ANN)

## 📌 Project Overview

This project aims to predict whether a bank customer will leave (churn) or stay with the bank using an Artificial Neural Network (ANN). The model analyzes customer data and identifies patterns that indicate churn behavior.

## 🎯 Objective

To build a predictive ANN model that helps banks identify customers who are likely to churn, allowing them to take proactive retention measures.

## 🛠 Technologies Used

* Python
* NumPy
* Pandas
* TensorFlow / Keras
* Scikit-learn

## 📊 Dataset Information

The dataset includes customer details such as:

* Credit Score
* Age
* Gender
* Balance
* Number of Products
* Tenure
* Estimated Salary
* Churn (Target Variable)

## 🧠 Model Architecture

* Input Layer (customer features)
* Hidden Layers (ReLU activation)
* Output Layer (Sigmoid activation for binary classification)

## ⚙️ Working Process

1. Data preprocessing (handling missing values, encoding categorical data)
2. Feature scaling
3. Splitting dataset into training and testing sets
4. Building ANN model
5. Training and evaluation

## 📈 Results

The model successfully predicts customer churn with good accuracy and helps in identifying high-risk customers.
